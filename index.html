    ./vim-install [TARGET_VERSION]

    build example

    $ git clone https://github.com/roadman/vimbuild.git
$ cd vimbuild
$ ./vim-install 7.4
$ echo 'export PATH="$HOME/vimbuild/build/7.4/vim74/src/:$PATH"' >> $HOME/.bashrc
